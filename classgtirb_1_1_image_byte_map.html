Set the byte map at the specified address.

Parameters
AThe address at which to store the data. Must be greater than the minimum address for this.
DataA pointer to the data to store. A + Data.size() must be less than the maximum address for this.
Returns
Will return true if the data can be assigned at the given Address, or false otherwise. The data passed in at the given address cannot overlap another memory region (overlays are not supported).

Data is written directly without any byte order conversions.
